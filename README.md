# 🚀 DevBrendow - Portfólio Profissional

Bem-vindo ao repositório do meu portfólio! Este projeto foi desenvolvido para centralizar meus trabalhos de **Desenvolvimento Web**, criação de **Sistemas Personalizados** e **Automações de Processos** (Google Apps Script).

---

## 🎨 Inspiração
Este projeto foi visualmente e estruturalmente inspirado no trabalho de **Gustavo Távora Cavalcante ([Gustavotcsi](https://github.com/Gustavotcsi))**, um desenvolvedor que admiro pela clareza e organização de seus projetos.

---

## 🛠️ Tecnologias
* **Frontend:** HTML5, CSS3 (Flexbox/Grid), JavaScript (ES6+).
* **Estilização:** Google Fonts (Poppins) & Font Awesome Icons.
* **Deploy:** GitHub Pages / Vercel.
